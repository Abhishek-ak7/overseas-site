# Flag Images Directory

This directory contains flag images for destination countries.

## Required Flag Images:
- australia.png
- canada.png
- usa.png
- uk.png
- germany.png
- france.png
- ireland.png
- cyprus.png
- new-zealand.png
- singapore.png
- uae.png

## Image Specifications:
- Format: PNG
- Dimensions: 64x48 pixels (4:3 ratio)
- Quality: High resolution for crisp display
- Background: Transparent or white

## Usage:
These flag images are used in the destination country pages to provide visual identification for each study abroad destination.

## Note:
You can download free flag images from sources like:
- Flagpedia (https://flagpedia.net/)
- Country Flags (https://countryflags.com/)
- Wikimedia Commons

Make sure to use royalty-free images or images with appropriate licensing.
